# Prompts для imagegen

## Как использовать

Ниже промпты уже заточены под визуальную новеллу. Они опираются на главы 1-4. Если какая-то деталь внешности не названа в тексте напрямую, она добавлена как рабочее дизайнерское решение для консистентности.

## Общий style prefix

Использовать в начале почти каждого промпта:

```text
cinematic semi-realistic visual novel illustration, painterly rendering, grounded Russian provincial realism, southern mountain village in spring, lilac bloom, emotional tension under beautiful daylight, expressive eyes, subtle film grain, clean silhouette design, no fantasy elements, no cyberpunk, no glossy anime look
```

## Общий negative prompt

```text
low detail, bad anatomy, extra fingers, duplicate limbs, crossed eyes, plastic skin, overexposed highlights, neon cyberpunk colors, fantasy armor, sci-fi props, text, watermark, logo, chibi, exaggerated anime proportions
```

## CG-подсказки для ключевых сцен

Если нужен не фон, а полноценная сюжетная иллюстрация:

- Для приезда Дианы: добавить `teenage girl in foreground looking through train window, spring sunlight, lilac bloom outside, expression of wary déjà vu`.
- Для сарая: добавить `teenage girl crouching with a black phone in hand, moonlight, fear mixed with analysis`.
- Для трибун: добавить `two teenagers sitting apart on empty stadium stands, blood-orange sky, confession scene, emotional distance turning into trust`.

## Рекомендации по консистентности

- Сначала сгенерировать 1 базовый концепт на каждого главного героя.
- После утверждения внешности делать expression sheets и CG только с формулой `same character design as previous concept`.
- Фоны генерировать отдельно от персонажей.
- Не перегружать кадр мелким реквизитом: в этой истории сильнее работают свет, воздух, пустота и дистанция между людьми.

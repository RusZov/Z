# Build Report

Дата: 2026-04-19

## Что собрано

- Реальный проект Ren'Py в `prototype/gorny_prototype/game/`.
- Первый играбельный акт по материалам из `script/branching_outline.md` и `script/game_variables.md`.
- 5 ключевых выборов игрока.
- 5 актовых исходов:
  - `A` — союз с Димой;
  - `B` — изоляция;
  - `C` — давление криминальной линии;
  - `D` — социальный щит Вики;
  - `E` — ранний провал и срыв маски.

## Что интегрировано

- Счётчики и флаги из сценарного пакета переведены в рабочие переменные Ren'Py.
- Фоны подключены через динамический резолвер:
  - проект ищет реальные изображения в `prototype/gorny_prototype/game/assets/backgrounds`;
  - отсутствие обязательного фона теперь считается ошибкой сборки.
- Спрайты персонажей подключены через динамический резолвер:
  - проект ищет PNG-файлы в `prototype/gorny_prototype/game/assets/characters/transparent`;
  - вшиты позы для Дианы, Димы, Миши, Вики и Марины;
  - отсутствие обязательного спрайта теперь считается ошибкой сборки.
- Подключён музыкальный слой первого акта:
  - проект ищет треки в `prototype/gorny_prototype/game/audio/bgm`;
  - в сценарий расставлены отдельные музыкальные состояния для школы, домашнего напряжения, саспенса, расследования и финала;
  - атрибуция сохранена в `prototype/gorny_prototype/game/audio/CREDITS.txt` и упомянута в `game/options.rpy`;
  - отсутствие обязательного трека теперь тоже считается ошибкой сборки.
- Встроен небольшой HUD прототипа с текущими значениями маршрута и ключевых параметров.
- Добавлен отдельный запускатель `prototype/launch-book.ps1` и ярлык на рабочем столе Windows с пользовательской иконкой.
